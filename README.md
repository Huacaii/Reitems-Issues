# Reitems
Minecraft Resourcepack, I reimagined the textures in Minecraft vanilla and some mods.

我的世界资源包，我重新设计了我的世界原版和一些模组内的纹理。

## 概述 / Overview

### Why am I making this resource pack?
Because some textures make the game look less distinctive for players, and some are even completely irrelevant, which makes me feel very uncomfortable, so I reimagined the textures in Minecraft vanilla and some mods, This resourcepack will be continuously updated and support more mods.

### 为什么我要做这个资源包？
因为有些材质让玩家看起来没有多大的辨识度，有些甚至直接无关，让我感到非常不适，所以我重新设计了 Minecraft 原版和一些模组里的材质，这个资源包会持续更新并支持更多模组。
